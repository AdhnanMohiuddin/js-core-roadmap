# 📝 Exercise 6: Objects (`6.objects.js`)

### Questions:
1. What is the difference between accessing a property using **Dot notation** vs **Bracket notation**?
2. How do you check if a specific property exists in an object using the `in` operator?
3. What is the **Object Property Shorthand** in ES6?
4. What happens when you use the **Spread Operator (`...`)** on two objects with the same key?

### Practice Task:
- Create a file named `solution6.js`.
- Create an object `smartPhone` with properties: `brand`, `model`, and `price`.
- Add a method `isExpensive` that returns `true` if the price is over 500.
- Use **Destructuring** to extract the `brand` and `model` into variables.
- Create a new object `discountedPhone` by spreading the `smartPhone` object and updating the `price`.
- Log the keys of the `smartPhone` object using `Object.keys()`.
