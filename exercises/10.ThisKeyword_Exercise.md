# 📝 Exercise 10: The 'this' Keyword (`10.this-keyword.js`)

### Questions:
1. In a regular function (not strict mode), what does `this` refer to in the global scope?
2. In a method, what does `this` usually refer to?
3. What is the difference between `.call()` and `.apply()`?
4. What does `.bind()` do that `.call()` doesn't?
5. Where does an **Arrow Function** get its `this` value from?

### Practice Task:
- Create a file named `solution10.js`.
- Create an object `video` with a property `title` and a method `play` that logs "`Playing ${this.title}`".
- Call `video.play()` and observe the output.
- Create a function `showBrand` that logs `this.brand`.
- Use `.call()` to execute `showBrand` with an object `{ brand: "Nike" }`.
- Use `.bind()` to create a new function `showAdidas` for the object `{ brand: "Adidas" }` and then execute it.
- Use a `setTimeout` with an arrow function to log `this.title` from inside the `video.play` method after a delay.
