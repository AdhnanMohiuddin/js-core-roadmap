# 📝 Exercise 8: Iteration Patterns (`8.iteration-patterns.js`)

### Questions:
1. What is the main difference between a `while` loop and a `do...while` loop?
2. Why is it generally recommended to use `for...of` for arrays instead of `for...in`?
3. What does `Object.entries()` return?
4. How do you stop a loop early when a certain condition is met?

### Practice Task:
- Create a file named `solution8.js`.
- Create an array of 5 names. Use a **`for...of`** loop to log each name in uppercase.
- Create an object `scores` with keys as student names and values as numbers.
- Use **`for...in`** to log the message: "Student [name] scored [value]".
- Re-write the above using **`Object.entries()`** and a `for...of` loop with destructuring.
- Use a `while` loop to find the first number divisible by 7 in a range of 1 to 50.
