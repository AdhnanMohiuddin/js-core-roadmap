# 📝 Exercise 12: Functional Iteration (`12.functional-iteration.js`)

### Questions:
1. What exactly is a **Higher-Order Function**?
2. What is a **Callback Function**?
3. Which array method should you use if you want to **transform** Every item in an array and get a new array?
4. What is the difference between `.find()` and `.filter()`?
5. How does `.reduce()` differ from methods like `.map()` or `.forEach()`?

### Practice Task:
- Create a file named `solution12.js`.
- Create an array `prices` with 5 different numbers.
- Use **`.map()`** to create a new array `taxedPrices` where each price is increased by 10%.
- Use **`.filter()`** to find all prices in the `prices` array that are greater than 100.
- Use **`.reduce()`** to calculate the total sum of the `prices` array.
- Create an array of strings `names`. Use **`.some()`** to check if any name starts with "Z".
- Use **`.find()`** to get the first name in the `names` array that has more than 5 characters.
