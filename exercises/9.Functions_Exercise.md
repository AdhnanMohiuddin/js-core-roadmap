# 📝 Exercise 9: Functions (`9.functions.js`)

### Questions:
1. What is the difference between a **Function Declaration** and a **Function Expression** regarding hoisting?
2. How do you define a **Default Parameter** for a function?
3. What are **Rest Parameters (`...`)** and when are they useful?
4. What is an **IIFE**, and what is its standard syntax?
5. Why are Arrow Functions usually avoided when writing object methods?

### Practice Task:
- Create a file named `solution9.js`.
- Write a **Function Declaration** `multiply` that takes two numbers and returns their product.
- Convert `multiply` into an **Arrow Function** with an implicit return.
- Create a function `describeUser` that takes `name`, `age` (default 18), and `...hobbies`.
- Log a string like: "Name is [name], age is [age]. Hobbies are: [hobbies list]".
- Write an **Anonymous Function** inside a `setTimeout` that logs "Done!" after 1 second.
