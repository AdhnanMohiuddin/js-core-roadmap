# 📝 Exercise 7: Control Flow (`7.control-flow.js`)

### Questions:
1. List the 6 **Falsy** values in JavaScript.
2. When would you prefer a `switch` statement over an `if...else if` chain?
3. How does the **Nullish Coalescing Operator (`??`)** differ from the **Logical OR (`||`)** operator?
4. What is the benefit of using **Guard Clauses** in a function?
5. What does **Optional Chaining (`?.`)** prevent?

### Practice Task:
- Create a file named `solution7.js`.
- Write a function `canDrive` that takes `age` and `hasLicense` (boolean).
- Use a **Guard Clause** to return "Invalid age" if age is less than 0.
- Use a **Ternary Operator** to return "Yes" if age >= 18 and hasLicense is true, otherwise "No".
- Use **Short-circuiting (`&&`)** to log "Access Granted" only if a variable `isVerified` is true.
- Use **Nullish Coalescing** to assign a default value of 10 to `userAttempts` if it is `null` or `undefined`.
